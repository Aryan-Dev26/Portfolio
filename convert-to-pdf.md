# How to Convert Resume to PDF

## Method 1: Using Browser (Recommended)

1. **Open the resume.html file** in your browser
2. **Press Ctrl+P** (or Cmd+P on Mac)
3. **Select "Save as PDF"** as destination
4. **Adjust settings:**
   - Paper size: A4
   - Margins: Minimum
   - Scale: 100%
   - Background graphics: ✅ Enabled
5. **Save as:** `Aryan_Sahu_Frontend_Developer_Resume.pdf`

## Method 2: Using Online Tools

1. **Upload resume.html** to:
   - HTML to PDF converter online
   - SmallPDF HTML to PDF
   - ILovePDF HTML to PDF

## Method 3: Using VS Code Extension

1. **Install "HTML to PDF" extension** in VS Code
2. **Right-click on resume.html**
3. **Select "Export to PDF"**

## Tips for Best Results:

- ✅ Use Chrome or Edge browser for best rendering
- ✅ Ensure all fonts load properly
- ✅ Check margins and page breaks
- ✅ Verify colors print correctly
- ✅ Test on different devices
